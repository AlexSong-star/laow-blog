## 我是怎么从「大海捞针」到「精准打窝」的

做俄罗斯 B2B 贸易，最头疼的事只有一个：**找不到对的人。**

一个真实的冶金厂，可能只有俄语网站、没有英文版、联系人藏在"联系销售部"的表单后面。传统的开发方式就是：**找协会、扫目录、发邮件、等回复**——效率极低，成功率更低。

这篇文章，记录我用 AI 工具把俄罗斯客户开发效率提升 10 倍的实战过程。不是概念验证，是真实跑通的生产流程。

---

## 工具一：HH.ru API——用招聘数据反向找客户

**痛点：** 直接找采购负责人难如登天，但找"正在招聘相关岗位"的公司容易得多。

**思路：** 俄罗斯最大招聘网站 HH.ru 上，一个公司如果长期招聘"冶金工程师""设备采购专员"，说明：
1. 这家公司还在运营
2. 有实际的生产/采购需求
3. 正在扩张——这时候联系最容易切入

**我的工作流：**
```
搜索关键词（俄语）：Валки прокатные（轧辊）、МНЛЗ（连铸机）
    ↓
HH.ru API 抓取招聘方信息：公司名、城市、联系人
    ↓
去重 + 归一化处理
    ↓
进入客户表格，等待背调
```

**实战数据：** 单次搜索 ~100 条线索，合格率约 40%，已经验证可用。

**局限：** 需要公司 INN（企业税号）才能查更多工商信息，数据有一定局限性。

---

## 工具二：Metaprom——工业B2B目录的爬虫挖掘

**Metaprom.ru** 是俄罗斯最大的工业 B2B 目录之一，涵盖了冶金、机械、铸造等领域的供应商和采购商。

**我的爬取策略：**
- 按行业关键词搜索：`https://metaprom.ru/search/?text=关键词`（俄语关键词需要 URL 编码）
- 解析每页公司列表：公司名、城市、官网、电话
- 进入官网抓取联系方式

**实战数据：** 
- 单次搜索 ~200-300 条记录
- 行业关键词选择决定数据质量
- 带官网的公司转化率更高

**踩过的坑：**
- 俄语 URL 编码错误会导致搜索无结果：`Валки` → `%D0%92%D0%B0%D0%BB%D0%BA%D0%B8`
- 频率过快会被临时封禁，爬取间隔 1-2 秒是安全线
- 部分公司网站已失效，需要二次验证

---

## 工具三：B2B Center——俄罗斯版「天眼查」的精准打击

**B2B Center**（b2b-center.ru）是俄罗斯头部的 B2B 交易平台，有完整的企业库和联系信息。

**为什么它有价值：**
- 有真实交易记录——能判断这家公司是活跃的供应商还是采购商
- 有 INN 可以做工商背调
- 有评分和交易历史

**局限：** 有 CAPTCHA 保护，不能无限制爬取。需要注册账号、手动操作或 OCR 解决。

## 我的完整工作流

```
第一阶段：线索挖掘（AI自动化）
    HH.ru API → 招聘数据 → 公司列表
    Metaprom 爬取 → 工业目录 → 公司列表
    B2B Center → 企业目录 → 公司列表
         ↓
第二阶段：数据清洗（AI辅助）
    去重 + 归一化
    官网有效性验证
    联系方式补全
         ↓
第三阶段：客户背调（进行中）
    INN 查询 → 工商信息
    官网评估 → 规模判断
    采购历史 → 需求验证
         ↓
第四阶段：个性化触达（计划中）
    AI 生成俄语开发信
    根据客户画像定制内容
    自动发送 + 跟进提醒
```

## 核心经验总结

1. **数据渠道组合拳**：单一渠道覆盖率有限，HH.ru + Metaprom + B2B Center 组合才能覆盖 80% 以上的目标客户
2. **俄语是门槛，也是壁垒**：能用俄语做搜索和数据处理，就已经打败了 90% 的中国竞争对手
3. **自动化是杠杆，人工判断是核心**：AI 做脏活，人做高价值决策，这才是正确的分工
4. **反爬是常态，调整节奏是智慧**：被封了就停，换时间再试，比硬冲聪明得多
